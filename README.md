# ultrasonic-16x2
Reads ultra sonic values and displays to a simple 16 x 2 LCD monitor.

### Hardware:

1. Arduino Uno / Mega 
2. 16 x LCD screen 
3. UltraSonic Sensor
4. Breadboard, Jumper Wires, other components.
